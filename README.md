# week6-app-jcnr


### To run this locally:

- `git clone` the repo
- `npm install` to install the necessary node packages.
- Create a `.env` file in the root folder and add API_KEY, DATABASE_URL and COOKIE_SECRET
- `npm run dev` to run the server
- visit `localhost:3000`
